# decentralised-ecosystem-prototype

<body>

<h1>Prerequisites</h1>
<p>Have configtxgen and cryptogen in $PATH</p>
<p>Navigate to the /basic-network folder with CLI</p>

<h1>To start</h1>
  <p>Run commands:</p>
    <ul>
        <li>./start.sh</li>
    </ul>
   <p>This spins up the distributed ledger and installs a chaincode contract</p>
<h1>To exit</h1>
  <p>Run commands:</p>
    <ul>
        <li>./teardown.sh</li>
    </ul>
</body>
